# HChat

HChat is a simple real-time chat application built with React, Node.js, Express, and Socket.IO.

This is my first full-stack project using JavaScript technologies.

## Features

* Real-time messaging
* Client-server communication
* Responsive chat UI

## Tech Stack

* React
* Node.js
* Express
* Socket.IO
* JavaScript

## Installation

```bash
git clone https://github.com/hamzaallaberdiyew-boop/HChat.git
cd HChat
npm install
```

## Run the Project

### Start server

```bash
cd server
npm start
```

### Start client

```bash
cd client
npm run dev
```
